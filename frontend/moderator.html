Moderator: Review Prompts


As a moderator, please review these prompts which users have tagged for delete.


Load Deleted Prompt will load a prompt from the recently_deleted collection for review. Delete Prompt will wipe this prompt from the recently_deleted collection forever. Approve will send the prompt back to the main collection, along with any edits you might have made. Save Edits will save the prompt in the recently_deleted collection for further action at a later time. Skip will skip the item and save it in the recently_deleted collection for further action at a later time.
